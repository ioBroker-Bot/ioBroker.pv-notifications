# ioBroker PV Notifications Adapter

Отправка Telegram-уведомлений о состоянии батареи PV-системы (полная, пустая, промежуточные уровни).

## Возможности

- 🔋 **Уведомление о полной батарее** при 100%
- ⚠️ **Уведомление о пустой батарее** при 0%
- 📊 **Промежуточные уровни** (20%, 40%, 60%, 80%) с уровнем заряда в % и кВт⋅ч
- 🌙 **Ночной режим** (настраиваемое время, по умолчанию: 23:00-06:00)
- 🤫 **Тихий режим** (настраиваемое время, по умолчанию: 12:00-15:00)
- 📈 **Дневная статистика** в настраиваемое время (по умолчанию: 22:00)
- 📅 **Недельная статистика** в настраиваемый день недели
- 📆 **Месячная статистика** (опционально) в настраиваемый день
- 🌤️ **Прогноз погоды** интеграция (требуется адаптер openweathermap)
- ⚡ **Рекомендации** при высокой выработке / высоком потреблении
- 📊 **Статистические данные** от адаптера sourceanalytix

## Зависимости

Для полной функциональности требуются следующие адаптеры:

| Адаптер | Описание | Обязательно |
|---------|----------|-------------|
| **telegram** | Отправка уведомлений | ✅ Да |
| **sourceanalytix** | Статистические данные (потребление, подача в сеть, мощность сети) | ✅ Да |
| **daswetter** или **openweathermap** | Прогноз погоды для рекомендаций | ❌ Опционально |

## Установка

### Через ioBroker Admin (GitHub)

```bash

https://github.com/sadam6752-tech/ioBroker.pv-notifications
```

### Через ioBroker CLI

```bash
iobroker url https://github.com/sadam6752-tech/ioBroker.pv-notifications
```

### Погода (опционально)

| Настройка | Описание | Пример (daswetter) | Пример (openweathermap) |
|-----------|----------|---------------------|-------------------------|
| Погода сегодня | Описание погоды сегодня | `daswetter.0.Day0.forecast.currentSymbol` | `openweathermap.0.forecast.0.text` |
| Температура сегодня (°C) | Температура сегодня | `daswetter.0.Day0.forecast.maxTemp` | `openweathermap.0.forecast.0.temp` |
| Погода завтра | Описание погоды завтра | `daswetter.0.Day1.forecast.currentSymbol` | `openweathermap.0.forecast.1.text` |
| Температура завтра (°C) | Температура завтра | `daswetter.0.Day1.forecast.maxTemp` | `openweathermap.0.forecast.1.temp` |

**Примечание:** Поля «Погода сегодня» и «Погода завтра» можно использовать альтернативно, если адаптер погоды предоставляет другие форматы. Для лучшей совместимости рекомендуем использовать поля `Weather Text`.

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|

| Порог ПУСТАЯ | SOC для «пустой» | `0` |

| Сброс ПУСТАЯ выше | Сброс если SOC > | `5` |

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|

| Мин. интервал ПУСТАЯ | Минуты между уведомлениями | `5` |

| Начало ночного режима | Время начала (формат: ЧЧ:ММ) | `23:00` |
| Конец ночного режима | Время окончания (формат: ЧЧ:ММ) | `06:00` |
| Игнорировать ночной режим для 0% | Всегда уведомлять при 0% | `true` |
| Включить тихий режим | Флажок для тихого режима | `false` |
| Начало тихого режима | Время начала (формат: ЧЧ:ММ) | `12:00` |
| Конец тихого режима | Время окончания (формат: ЧЧ:ММ) | `15:00` |

### Статистика

| Настройка | Описание | По умолчанию |

## Примечание о месячной и недельной статистике

**Важно:** Адаптер автоматически сохраняет данные за прошлый месяц и прошлую неделю в states.

### Месячная статистика

- Месячная статистика отправляется в **настраиваемый день** (по умолчанию: 1-е число месяца)
- Адаптер **автоматически сохраняет** текущие месячные данные перед сбросом статистики
- Статистика использует **сохраненные данные** из states `statistics.lastMonth*`
- **Настройка:** Убедитесь, что месячная статистика отправляется **после последнего дня месяца** (например, 1-го в 09:00)

### Недельная статистика

- Недельная статистика отправляется в **настраиваемый день недели** (по умолчанию: Понедельник)
- Адаптер **автоматически сохраняет** текущие недельные данные перед сбросом статистики
- Статистика использует **сохраненные данные** из states `statistics.lastWeek*`
- **Настройка:** Установите день недели (0=Пн, 1=Вт, ..., 6=Вс)

## Ночной режим и Тихий режим

### Ночной режим (настраиваемый)

В период **с 23:00 до 06:00** (настраивается) следующие уведомления подавляются:
- ❌ Батарея ПОЛНАЯ (100%)
- ❌ Промежуточные уровни (20%, 40%, 60%, 80%)

Следующее уведомление отправляется **всегда**:
- ✅ Батарея ПУСТАЯ (0%) – если включено «Игнорировать ночной режим для 0% батареи»

### Тихий режим (настраиваемый)

В период **с 12:00 до 15:00** (настраивается) **все** уведомления подавляются:
- ❌ Батарея ПОЛНАЯ (100%)
- ❌ Батарея ПУСТАЯ (0%)
- ❌ Промежуточные уровни (20%, 40%, 60%, 80%)

**Примечание:** Тихий режим подавляет все уведомления, включая 0% батареи. Используйте его для периодов, когда вы не хотите, чтобы вас беспокоили вообще (например, тихий час, совещания).
